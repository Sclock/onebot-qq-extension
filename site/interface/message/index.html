    表情

    字段名 数据类型 说明
    字段1 int64 获取的事件数量上限，0 表示不限制
    字段2 string 没有事件时要等待的秒数，0 表示使用短轮询，不等待

    "type": "type",
    "data": {
        "user_id": "1234567"
